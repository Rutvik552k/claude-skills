# Skills Project

This project contains 9 domain knowledge skills built from 14 technical books.

## Available Skills

Skills are in the `./` directory. Each subdirectory is a skill with a `SKILL.md` file:

- `dsa-fundamentals/` — Data structures, algorithms, sorting, complexity analysis
- `software-testing/` — Boundary value, equivalence class, path testing, integration, regression
- `discrete-math-cs/` — Proofs, induction, number theory, RSA, graph theory, probability
- `offensive-security/` — Buffer overflows, shellcode, network attacks, countermeasures (educational)
- `nlp-deep-learning/` — NLP pipelines, embeddings, RNNs/LSTMs, attention, seq2seq, PyTorch
- `container-orchestration/` — Podman, Kubernetes, pods, deployments, services, networking
- `financial-ai/` — Financial argument mining, multi-agent finance, blockchain, portfolio optimization
- `deep-learning-mathematics/` — ANN theory, optimization, generalization, SVD/PCA, PINNs
- `diffusion-generative-models/` — DDPMs, score SDEs, text-to-image, text-to-video generation

## Usage

When working on a task in one of these domains, read the relevant `SKILL.md` for topic coverage and consult `references/` for deep dives.
